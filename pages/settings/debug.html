Benchmarking

Benchmarking has been removed from Eclipse as of version 2.0.0. This data was going to be used to limit underpowered devices from certain features, but other methods will probably be used to do this.

If you are having performance issues, please ensure your battery wear is not bad enough to enable throttling and that power-saving features are disabled.
